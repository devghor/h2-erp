# H2 ERP

A multi-tenant ERP system built with **Laravel 12** (backend) and **Nuxt 3** (frontend). Features a Filament-powered admin panel, role-based access control via Spatie Permission, and tenant management with Stancl Tenancy.

## Stack

- **Backend:** Laravel 12, Filament 4, Spatie Laravel Permission, Stancl Tenancy
- **Frontend:** Nuxt 3, Vue 3
- **Infrastructure:** Docker

## Modules

- User Access Management (users, roles, permissions)
- Customer management
- Multi-tenancy

## Getting Started

```bash
docker compose up -d
```
